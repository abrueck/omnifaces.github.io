    org.omnifaces.util.Faces.includeFacelet(UIComponent, String)
    Moved to Components.includeFacelet(UIComponent, String). Please use it instead.
    org.omnifaces.util.Events.removeCallbackPhaseListener()
    Because it only removes the one being set by Events.setCallbackPhaseListener(PhaseListener) which is currently deprecated.
    org.omnifaces.util.Events.setCallbackAfterPhaseListener(PhaseId, Callback.Void)

    Because it allows only one callback phase listener to be set. Please use Events.addCallbackPhaseListener(PhaseListener) instead.
