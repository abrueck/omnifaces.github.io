org.omnifaces.util

Class Beans

  • java.lang.Object
    • org.omnifaces.util.Beans

  • public final class Beans
extends Object
    Collection of utility methods for the CDI API with respect to working with BeanManager.

    If you need a dependency-free way of obtaining the CDI managed bean instance (e.g. when you want to write code which should also run on Tomcat), use BeanManager instead.

    Since:
    1.6.1
    Author:
    Bauke Scholtz
