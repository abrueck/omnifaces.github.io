    @NamedEvent(shortName="preInvokeAction")
public class PreInvokeActionEvent
extends ComponentSystemEvent
    Use this event to have a hook on a listener method during the beforephase of the PhaseId.INVOKE_APPLICATION. This event is supported on UIViewRoot, UIForm, UIInput and UICommand components.

    This event is particularly helpful as a replacement of <f:event type="preRenderView"> and also provides the possibility to invoke multiple action listeners on a single UIInput and UICommand components on an easy manner. 

     <f:event type="preInvokeAction" listener="#{bean.preInvokeAction}" />
